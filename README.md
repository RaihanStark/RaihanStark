## 🌱 The mission

I'm betting my career on a simple idea: **good software should be a public good, not a subscription.**

Everything I build is open source and stays open source — no bait-and-switch to a paid version, no feature held hostage behind a license key, no telemetry you didn't ask for. My goal is to do this **full time, funded entirely by [GitHub Sponsors](https://github.com/sponsors/RaihanStark)**, so the work answers to the people who use it instead of to a pricing page.

If a tool I made saves you time, sponsoring even a little is what keeps the next tool free for everyone.
